Ejemplo de Implementación del Menú Desplegable de Perfil

Tu Sitio Web

Perfil
Tu perfil Configuración Cerrar sesión

Instrucciones de Uso:

  1. Incluye el archivo CSS profile-dropdown.css en tu página
  2. Incluye el archivo JavaScript profile-dropdown.js en tu página
  3. Incluye Font Awesome para los iconos
  4. Copia la estructura HTML del menú desplegable
  5. Personaliza los enlaces según tus necesidades
  6. Funciona con CLICK - Haz click en "Perfil" para abrir/cerrar el menú

Estructura HTML:

<div class="profile-menu">
    <a href="#" class="cta-button">
        <i class="fas fa-user"></i> Perfil
    </a>
    <div class="dropdown">
        <a href="#">
            <i class="fas fa-user-edit"></i> Tu perfil
        </a>
        <a href="#">
            <i class="fas fa-cog"></i> Configuración
        </a>
        <a href="../Maqueta/login.html">
            <i class="fas fa-sign-out-alt"></i> Cerrar sesión
        </a>
    </div>
</div>

<!-- No olvides incluir el JavaScript -->
<script src="../JS/profile-dropdown.js"></script>
